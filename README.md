# ログイン機能付き TODO アプリ

このプロジェクトは、基本的なフロントエンド技術を学習するために作成した、ログイン機能とTODO管理機能を備えたWebアプリケーションです。

## 🚀 機能一覧
- **ログイン機能**: デモ用のユーザー情報でログイン・ログアウトが可能。
- **タスク管理**:
  - タスクの追加（Enterキー対応）
  - 完了状態の切り替え（打ち消し線表示）
  - タスクの削除
- **データの保存**: ブラウザを閉じてもデータが保持される LocalStorage を活用。

## 🛠 使用技術
- HTML / CSS
- JavaScript
